    Namespace BlackMassSoftware.FloatingTextEngine.Behaviors

    Classes

    BlinkBehavior

    This behavior causes the text to turn on and off simulating a blink. It takes a blink speed and a total duration.

    ChangeFontSizeBehavior

    This behavior will set this floating texts specific size. This is a substitute for changing the global size through the FloatingTextEngine inspector window.

    ColorGradientBehavior

    This behavior will take 2 colors and lerp between them.

    ExpandShrinkBehavior

    This behavior will scale up and then down for the specified amount of times.

    FadeInOutBehavior

    This behavior will fade in and fade out at the specified speed (seconds for full cycle) over time.

    FadeOutBehavior

    This behavior will fade out the text over the specified duration.

    FloatingBehaviorEngine

    This class handles all the behaviors each spawned floating text object has. It will update all behaviors and remove them when finished until no more behaviors are left. This class also handles the objects return to the object pool.

    FloatingTextBehaviors

    This wrapper class hides the "new" calls for each behavior. This needs to be updated when a new behavior is implemented. Pass these methods into the .With() function from the FloatingBehaviorEngine class.

    IFloatingBehavior

    Interface class for all floating text behaviors. Implement this class to define new behavior components.

    MoveUpArcBehavior

    This behavior moves up and in an arc. The arc is determined by the random x value within the "arcSpread" value.

    MoveUpBehavior

    This behavior moves up in the Y axis.

    MoveUpExpandBehavior

    This behavior moves up in the Y axis then expands after movement has finished.

    PopBehavior

    This behavior scales up quickly and then slowly scales down over time.

    ScaleToBehavior

    This behavior scales to the value passed in.

    ShakeBehavior

    This behavior will cause the floating text to shake for a specified duration.

    ShakeXBehavior

    This behavior causes the floating text to shake along the x axis.

    WobbleBehavior

    This behavior causes the floating text to rotate back and forth on the z axis.
